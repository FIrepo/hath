There are two roles in a Housie/Bingo/Tambola game :
  • Organizer : Controls the game, allows players to join, issues tickets and draws numbers. This is 'running' a game in housie@home. To organize a game, select 'Run' option on the housie@home home page. As a organizer, you do following :
    • Create a game
    • Let players know the Game Tag and Game password (For those who can join online)
    • Print tickets (for players who cannot join online)
    • Draw numbers
    • Give prizes to the winners !!!!
  • Player : Joins game, gets tickets, follows game as numbers are drawn and potentially, win the game. To play, select the 'Play' option on the home page :
    • Join a game
    • Get tickets
    • View your tickets and follow the progress of the Game
    • Win prizes !!!!
Creating/Joining new game needs below information :
  • Game Tag : It is the unique identifier for a game.
  • Admin Password : This the password for organizer to access and run the Game once again, if by any chance the game is interrupted.
  • Game Password : This is needed by players alongwith Game Tag to join a Game.
  • Player name : Name of the player - has to be unique for the game.
  • Player password : This is an optional password set by a Player so that someone else with the same is not taking over the Player's partcipation.
Continuing an existing game :
  • If by any reason, the page is refreshed in browser, the users will see Create / Join panel again.
  • In such a case, pressing on 'Continue an existing game' will show a list of games Origanizer/Player have already started/joined.
  • Selecting one from the list and press 'Continue' button will let you continue from where you left.
  • If for any reason, the browser is closed, and a new browser window is opened, the games list will be empty and 'Continue' option will not work.
  • In such case, user will have to use Create/Join options and enter the Game tag, passwords etc.
  • If a game with the same Game tag and passwords etc. is found, then user can continue the game.
Once an organizer creates a game, several tabs are visible. Information about the game is organized under these.
  • Print Tickets : Tickets for offline players can be printed using this feature. Tickets are displayed in a new window which then have to be printed using Browser print feature.
  • Draw Numbers : This is for drawing numbers of the Game
  • View Status : This displays a grid of numbers with numbers already drawn in green background and those pending in red background.
  • Game Stats : This displays a summary of game - list of players, tickets issued, how many numbers are still pending etc.
For players, below options are available :
  • Get Tickets : Players can get the tickets for the game here. A Player has a choice to discard a ticket or confirm it. Confirming is needed of the ticket is part of the game.
  • View Tickets : Players can view the tickets issued to them here. If a number drawn is on the tickets, it is automatically highlighted on the tickets.
  • View Status : This displays a grid of numbers with numbers already drawn in green background and those pending in red background.
